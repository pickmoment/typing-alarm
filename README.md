# Typing Alarm

시간을 타이핑해서 바로 알람을 등록하는 크롬 확장 프로그램입니다.
복잡한 설정 없이 `9`, `09:30`, `20분후`, `내일 9시` 같은 입력으로 빠르게 알람을 만들 수 있습니다.

## 주요 기능

- 다양한 시간 입력 지원
- 상대 시간 입력 지원 (`20분후`, `1시간 10분후`, `1분 10초뒤`)
- 날짜+시간 입력 지원 (`내일 9시`, `2/12 09:30`, `2026-02-12 09:30`)
- 여러 알람 한 번에 등록 (콤마 또는 줄바꿈)
- 알람 메시지 지정 (`09:30 물 마시기`)
- 반복 옵션 지원 (한 번, 매일, 평일, 주말, 요일 선택)
- 지정 시간보다 미리 울리기 (예: 10분 전)
- 자주 쓰는 문구 저장/재사용
- 알람 소리 지속 시간/간격 설정

## 설치 방법 (Chrome)

1. 이 폴더를 로컬에 준비합니다.
2. 크롬 주소창에 `chrome://extensions` 입력 후 이동합니다.
3. 우측 상단 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
5. 이 프로젝트 폴더를 선택합니다.
6. 확장 목록에 `Typing Alarm`이 나타나면 설치 완료입니다.

## 빠른 시작

1. 브라우저 툴바에서 `Typing Alarm` 아이콘을 클릭합니다.
2. 상단 입력창에 시간(또는 시간+메시지)을 입력합니다.
3. `Enter`를 눌러 알람을 추가합니다. (`Shift + Enter`는 줄바꿈)
4. 아래 `알람 목록`에서 시간/메시지/반복/활성 상태를 수정 후 `저장`합니다.

## 입력 예시

### 기본 시간

- `9`
- `21`
- `930`
- `0930`
- `9:30`
- `9:30am`
- `9시30분`
- `9:30:15`
- `93015`

### 상대 시간

- `20분후`
- `1시간후`
- `1시간 20분후`
- `1분10초후`
- `1분10초뒤`

### 날짜 + 시간

- `오늘 9시`
- `내일 9:30`
- `모레 18시`
- `2/12 9:30`
- `2026-02-12 09:30`
- `2026년2월12일 09:30`

### 메시지 포함

- `09:30 스트레칭`
- `20분후 회의 준비`
- `내일 8:00 아침 약`

## 자주 쓰는 기능

### 여러 알람 한 번에 등록

입력창에서 콤마 또는 줄바꿈으로 여러 개를 입력하면 한 번에 등록됩니다.

예시:

```text
9:00 업무 시작
11:30 점심 준비
14:00 물 마시기, 16:30 정리
```

### 미리 예약 적용

- `지정 시간보다 미리 예약 적용`을 켜고 분/초를 입력하면, 지정된 시간보다 먼저 알람이 울립니다.
- 이 기능은 `09:30` 같은 **지정 시간 입력**에만 적용됩니다.
- `20분후` 같은 **상대 시간 입력**에는 적용되지 않습니다.

### 알람이 울릴 때

- 브라우저 알림(Notification)이 표시됩니다.
- 팝업 창이 열리고 소리가 반복 재생됩니다.
- `알람 끄기` 버튼으로 즉시 중지할 수 있습니다.
- 소리는 설정한 `지속 시간`이 지나면 자동으로 종료됩니다.

## 권한 안내

이 확장 프로그램은 아래 권한을 사용합니다.

- `alarms`: 알람 예약/실행
- `storage`: 알람 및 설정 저장
- `notifications`: 브라우저 알림 표시
- `offscreen`: 백그라운드 사운드 재생

## 문제 해결

- 알람 창이나 알림이 보이지 않으면 브라우저 알림 권한이 허용되어 있는지 확인해 주세요.
- 소리가 나지 않으면 탭/시스템 음소거 상태를 확인해 주세요.
- 입력이 인식되지 않으면 예시 형식에 맞춰 다시 입력해 주세요.
